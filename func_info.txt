function BossShow
整套boss展示动画(与BossAnimEnd配合使用)
包含：
    SetAllAI;0
    ShowAllMonster;0
    ShowAllHero;0
    ShowAllItem;0
    ShowAllUI;0
    CreateShowBoss;boss=31000405
    MoveCameraToBossCamera;1;easeInOut
    ShowFigthCameraObj;0
    PlayBossAnim;43
boss boss的id

function BossAnimEnd
整套boss展示动画结束
包含：
    ShowFigthCameraObj;1
    MoveBackCamera:0
    DeleteShowBoss
    SetAllAI;1
    ShowAllMonster;1
    ShowAllHero;1
    ShowAllItem;1
    ShowAllUI;1

function EndPose
结束动画
包含：
    SetAllAI;0
    ShowAllMonster;0
    ShowAllHero;0
    ShowAllItem;0
    CreateTeam;pos_name=
    ShowFigthCameraObj;0
    PlayTeamEnd;anim_id=
anim_id 动画id(详见action_list.txt)
pos_name 位置旋转大小读取map_info中burchhero里的pos_name变量
show_hero_num 显示英雄数量

function EndPoseFinish
结束动画恢复
包含：
    StopTeamEndAnim

function BossDead
boss死亡慢动作
包含：
    SetAllAI;0
    PlayBossDeadAnim;;;
2 死亡动画慢放时间
3 进入下个阶段时间

function BeginAnim
开场镜头展示
包含：
    ShowAllUI;0
    SetAllAI;0
    RecordSceneCameraInfo
    SetCamera;lookat=1;
    Wait;1
    MoveBackCamera;lookat=1;3;easeInOut
    ShowAllUI;1
    SetAllAI;1
lookat 是否看向目标1是
2 时间
3 缓动类型

function ScreenPlay;
剧情动画(与ScreenPlayOver配合使用)
包含：
    SetAllAI;0
    ShowAllUI;0
    ShowAllMonster;0
    ShowAllHero;0
    ShowAllItem;0
    ShowFigthCameraObj;0
    PlayScreenplay;obj=;pos=;rot=;anim_name=
obj 剧情需要对象文件名列表
pos 所有对象位置
rot 所有对象旋转
anim_name 所有对象动画名字
rollback 只有0时为不等待动画执行完就回调

function ScreenPlayOver
剧情动画结束
包含：
    SetAllAI;1
    ShowAllUI;1
    ShowAllMonster;1
    ShowAllHero;1
    ShowAllItem;1
    ShowFigthCameraObj;1

function CreateObj
创建一个对象
obj 文件名
id 唯一id号
pos 创建位置
rot 创建旋转

function PlayObjAnim
播放对象动画
xxx=anim_name;xx2=anim_name;=0;
rollback 只有0时为不等待动画执行完就回调

function DeleteObj
删除对象
list 文件列表

function Wait
等待一段时间
2 等待时间

function SetAllAI
是否开启所有人ai
2 是否开启

function ShowFigthCameraObj
是否显示战斗相机
2 是否开启

function ShowAllMonster
是否显示所有怪节点
2 是否开启

function ShowAllHero
是否显示所有英雄节点
2 是否开启

function ShowAllItem
是否显示所有道具节点
2 是否开启

function ShowAllUI
是否显示ui节点
2 是否开启

function CreateShowBoss
创建boss
会保存怪物对象在scene_entity中
boss 怪物id

function PlayBossAnim
播放保存在scene_entity的展示动画
2 动作id(详见action_list)

function MoveCameraToBossCamera
镜头缓动到展示动画镜头位置
会保存当前镜头和镜头的fov，位置，旋转，镜头状态信息
2 时间
3 缓动类型

function MoveBackCamera
将镜头缓动设置到保存的fov，位置，旋转，状态
2 时间
3 缓动类型
lookat 是否看向目标1是

function DeleteShowBoss
删除scene_entity对象

function PlayBossDeadAnim
播放boss死亡展示动画
2 慢放时间
3 进入下个阶段时间

function CreateTeam
创建结束队伍人物并放在指定位置
pos_name 位置旋转大小读取map_info中burchhero里的pos_name变量

function PlayTeamEnd
播放队伍结束动画
anim_id 动画id(详见action_list.txt)

function RecordSceneCameraInfo
记录战斗摄像机的位置、旋转、状态、fov信息

function SetCamera
设置战斗相机
（每个参数不填则保持原值）位置旋转大小读取map_info中burchhero里的begin_camera
fov 镜头fov值
lookat 是否看向目标1是
pos_name 位置旋转读取map_info中burchhero里的pos_name变量

function PlayScreenplay
播放剧情动画
rot、pos、anim_name是对所有对象统一设置的，不能分开设置	
obj 剧情需要对象文件名列表
pos 所有对象位置
rot 所有对象旋转
anim_name 对象动画名字
rollback 只有0时为不等待动画执行完就回调

function CameraMoveTo
相机移动至目标点
time 移动所需时间
ease 缓动类型
pos 镜头结束位置
rot 镜头旋转角度
lookat 是否看向目标1是

function BasicCameraMoveTo
仅用于关卡结束的敌方移动镜头
speed 速度(unity距离/秒）
ease 缓动类型
pos 镜头结束位置
rot 镜头旋转角度
lookat 是否看向目标1是

function PlayVideo
播放剧情视频
2 视频路径

function PlayUiAudio
播放声音文件
audio_id 音效id

function PlayTransitionUI
白屏过场（胜利/失败过场）

function PlayEndSound;
结束队伍展示声音

function HideSceneEffect
隐藏场景特效

function BlackFadeEffectUI
黑屏过场
type 黑屏类型FadeIn,FadeOut,KeepBlack
time 默认1s
text 文本内容

function ChangeBackAudio
切换背景音乐
audio_id 音乐id

function CreateObstacle
创建动态阻挡
list mapinfo中MonsterBorn位置名字列表
